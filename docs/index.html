三种对战模式

无论身边有没有网络，总有一种方式适合你和你的对手。

Hot Seat

热座模式

  • 两名玩家共用一台设备轮流操作
  • 无需任何网络连接
  • 面对面对战，适合朋友聚会
P2P LAN

P2P 局域网联机

  • 同一局域网内直连对战
  • 点对点连接，低延迟无需服务器
  • 卡牌资源在玩家之间同步传输
Server

服务器联机

  • 通过服务器与远程玩家对战
  • 跨网络匹配，随时随地开局
  • 稳定的房间与连接管理
